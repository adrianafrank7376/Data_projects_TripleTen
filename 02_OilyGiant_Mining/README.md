## üìà Methodology

1. **Exploratory Data Analysis**
   - Analyzed reserve distributions across regions
   - Identified outliers and data quality issues
   - Compared regional characteristics

2. **Predictive Modeling**
   - Built Linear Regression model for each region
   - Predicted oil reserves for potential wells
   - Validated model accuracy using RMSE

3. **Profit Calculation**
   - Selected top 200 wells per region
   - Calculated revenue based on reserves and oil price
   - Subtracted investment costs for net profit

4. **Risk Analysis (Bootstrapping)**
   - Generated 1000 bootstrap samples per region
   - Calculated profit distribution
   - Determined 95% confidence intervals
   - Assessed probability of losses
